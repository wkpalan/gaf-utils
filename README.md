Boilerplate for a Python Package

## Package

Basic structure of package is

```
├── README.md
├── gaf
│   ├── __init__.py
│   └── gaf.py 
├── tests
|   ├── __init__.py
|   ├── test.gaf
│   └── test_gaf.py
├── requirements.txt
├── setup.cfg
└── setup.py
```

## Requirements

Package requirements are handled using pip. To install them do

```
pip install -r requirements.txt
```
